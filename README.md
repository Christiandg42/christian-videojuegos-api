# API REST - Videojuegos favoritos de Christian

Proyecto desarrollado con Spring Boot para gestionar videojuegos favoritos.

## Tecnologías
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Maven

## Funcionalidades
- CRUD completo
- Búsqueda por género
- Búsqueda por plataforma
- Búsqueda por título
